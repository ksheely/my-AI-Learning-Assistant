# AI Learning Assistant

A simple AI-powered web app that helps users explore career paths, identify skill gaps, and generate project ideas based on their background or interests.

## Features

- Paste in a resume summary, interests, or experience
- Generate:
  - 2–3 possible career paths
  - missing skills to develop
  - 2 project ideas to build experience
- Clean web interface using FastAPI and HTML

## Tech Stack

- Python
- FastAPI
- OpenAI API
- Jinja2
- HTML/CSS/JavaScript
